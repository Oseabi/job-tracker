# 📋 Job Tracker

> Your job hunt, organised. No more sticky notes.

A Kanban-style job application tracker that lets you drag applications across stages — from Applied all the way to Offer (or the dreaded Rejected).

---

## 🔥 Why I built this

I was applying to jobs and tracking everything in a notes app like a person who enjoys chaos. One missed follow-up too many and I decided to build something proper.

**Problem:** Job hunting is stressful enough without losing track of where every application stands.  
**Solution:** A visual Kanban board where every application has a home and you can drag it forward as things progress.

> "Built this app while actively needing this app. The irony was not lost on me." 😂

---

## ✨ Features

- Kanban board with 4 stages — Applied, Interview, Offer, Rejected
- Drag and drop cards between columns
- Add company, role, work type and notes per application
- Live stats dashboard
- Dark and light mode with saved preference
- All data saved to localStorage — survives page refresh
- Clean, professional Arctic Blue design

---

## 🛠️ Built with

- Vanilla JavaScript — no frameworks
- HTML5 Drag and Drop API
- CSS3 with custom properties
- localStorage for data persistence
